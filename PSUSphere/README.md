# PSUSphere

**Project name:** PSUSphere

**Short description:**
A Django-based web application for managing student organizations and events at a university. Provides models for organizations, members, and associated data with an administrative interface.

**List of features:**
- CRUD operations for student organizations and related models
- Django admin integration for easy data management
- Data loading scripts and fixtures
- Requirements file for environment reproducibility
